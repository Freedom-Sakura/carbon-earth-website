低碳行动

从日常生活的小改变开始，创造美好的绿色未来

低碳生活行动指南

🚴

绿色出行

减排潜力： 每年可减少1-3吨CO₂

  • ✓ 使用公共交通、骑自行车或步行
  • ✓ 选择电动汽车或混合动力车
  • ✓ 拼车出行，减少单人出行频率
  • ✓ 减少飞行，考虑虚拟会议
  • ✓ 支持共享出行服务
⚡

节能居家

减排潜力： 每年可减少0.5-1.5吨CO₂

  • ✓ 安装LED灯泡，节能70%电力
  • ✓ 使用智能温度控制，优化供暖/冷却
  • ✓ 绝缘改善，减少能源损耗
  • ✓ 使用可再生能源（太阳能、风能）
  • ✓ 关闭不使用的电子设备
🥗

改变饮食

减排潜力： 每年可减少1-2吨CO₂

  • ✓ 减少肉类消费，采用素食或半素食
  • ✓ 选择本地季节性食物
  • ✓ 避免过度包装的食品
  • ✓ 减少食物浪费，计划购物
  • ✓ 支持有机和可持续农业
♻️

循环消费

减排潜力： 每年可减少0.3-0.8吨CO₂

  • ✓ 购买二手商品，延长产品寿命
  • ✓ 修理而不是更换物品
  • ✓ 分享、租赁而不是购买
  • ✓ 正确分类和回收废物
  • ✓ 选择可持续和环保品牌
🌱

绿化环境

减排潜力： 每年可吸收0.2-0.5吨CO₂

  • ✓ 种植树木和植物
  • ✓ 创建家庭菜园或阳台花园
  • ✓ 支持森林保护项目
  • ✓ 参加社区绿化活动
  • ✓ 推广办公室室内植物
